  #### 🛠️ Open source

  - **[Nimbus](https://github.com/ant-caor/Nimbus)**: Cloud Run-first
  caching for Go. An in-process L1, a versioned Redis L2 as the source
  of truth, and a Pub/Sub bus that keeps per-instance caches coherent
  across ephemeral, autoscaling instances.

  - **[Lang-Lab](https://github.com/ant-caor/Lang-Lab)**: Reproducible
  benchmarks for backend languages. It counts CPU instructions
  instead of wall-clock time, under a uniform QEMU+insn engine, so the
  numbers are deterministic and reproducible across machines. 12
  languages and a C baseline, a 15-benchmark suite, auto-refreshed on
  CI when a language ships a new version.
